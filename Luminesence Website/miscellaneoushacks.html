Here is a code snipit that you can copy
into the F12 menu to rename the current tab!

document.title = tab_name; // < 3 echo!


Tab Icon Change

Hate the icon of a tab? Just copy this code
into the F12 box and than click enter!
Paste the wanted image address into the pop-up.

var icon_href = window.prompt("Image adress for tab icon")
var link = document.querySelector("link[rel~='icon']");
if (!link) {
link = document.createElement('link');
link.rel = 'icon';
document.getElementsByTagName('head')[0].appendChild(link);
}
link.href = icon_href;
